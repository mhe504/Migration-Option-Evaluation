Task Descriptor: Query Logging
Capture queries sent between the system and the database to determine workload.
Based on Method Task: Query Logging
Main Description
Query Logging is a method for providing detailed workload information by intercepting queries sent from the software system to the database and recording them. Several third-party tools can perform this task, the output should be list of SQL queries and the time they were made.
