Task: Select Simulation Goal
Select a goal for the simulation, i.e., what question will the simulation answer.
Main Description

The following cloud database migration options are supported:

- New cloud database instance type(s)
- Cloud platform, region, or availability zone
- Database deployment on a Database-as-a-Service vs. a Virtual Machine.
- Database size growth
- Database workload
- Performance of the existing database
- Migration Infrastructure (e.g., is a middleware being used, like the Amazon Database Migration Service)
- A downtime or zero-downtime migration

A goal should be created using some conbination of the above options. It should specifiy the information the organisation is trying to obtain.
