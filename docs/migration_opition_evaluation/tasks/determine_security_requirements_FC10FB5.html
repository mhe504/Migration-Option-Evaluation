Task: Determine Security Requirements
The security requirements for the new cloud database and the migration of existing data to it should be identified.
Purpose
Indenification of security requirements which require additonal (chargable) cloud inrastructure, or affect performance.
Main Description
The security requires will be depdant on the data, database type, and the location of the source and target databases. Data can be secured at-rest in the new cloud database by using a Virtual Private Cloud or storage encryption. Data can be secured during transfer with a VPN or by using a database's own encyrption protocol.
