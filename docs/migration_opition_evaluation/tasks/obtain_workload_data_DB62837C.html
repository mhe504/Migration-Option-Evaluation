Task: Obtain Workload Data
Obtain the workload data from the previously identified sources. It must be processed to produce CSV/Text file in the MigSim format.
Key Considerations
MigSim CSV format, Line 1: System Name, Scope (String), Entity Name. Line 2 to n: Measurement Period Alias (String), Measurement Period Start (Timestamp; yyyy-mm-dd hh:mm:ss), Measurement Period End (Timestamp; yyyy-mm-dd hh:mm:ss), Entity Count (Integer),  Entity Reads (Integer), Entity Writes (Integer), Unused Entities (Integer), Database Size (Double).
